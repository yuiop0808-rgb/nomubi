  1. 인쇄: 위 "🖨 인쇄하기" 버튼 탭 → 프린터 선택 후 인쇄
  2. Safari 공유: 하단 공유(□↑) 버튼 → "프린트" 선택

  4. PDF 저장: "🖨 인쇄하기" 버튼 → 프린터 선택 화면에서 두 손가락으로 미리보기 화면을 확대하면 PDF로 저장 가능
